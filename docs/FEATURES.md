# 기능 설명

## 1. Home 화면

Home 화면에서는 좋아요 수를 기준으로 인기 도서 목록을 보여줍니다.

### 기능

- 좋아요 수 기준 도서 정렬
- 인기 도서 목록 표시
- 전체 목록 페이지 이동
- 도서가 없을 경우 빈 상태 안내 문구 표시

## 2. 도서 목록 화면

도서 목록 화면은 `/books` 데이터를 기반으로 카드 목록을 렌더링합니다.

### 기능

- 도서 카드 목록 표시
- 도서 제목 / 작가 기준 검색
- 검색 결과 없음 안내
- 등록된 도서 없음 안내
- 카드 클릭 시 상세 모달 표시
- 상세 모달에서 좋아요 / 조회수 / 수정 / 삭제 기능 제공

## 3. 도서 등록 화면

도서 등록 화면은 신규 도서 정보를 입력하고 저장하는 기능을 제공합니다.

### 입력 항목

- 도서 제목
- 작가 이름
- 도서 내용
- 표지 이미지

### 이미지 미리보기

이미지 미리보기 버튼을 누르면 OpenAI 이미지 생성 API를 통해 표지 이미지를 생성합니다.

중요한 점은 이미지 미리보기와 등록 동작이 분리되어 있다는 것입니다.

```text
이미지 미리보기 클릭
→ 이미지 생성만 수행

등록하기 클릭
→ 현재 입력값과 현재 이미지로 도서 등록
```

## 4. 도서 수정 화면

도서 수정 화면은 `/books/:id`로 기존 도서 데이터를 조회한 뒤 수정 폼에 반영합니다.

### 기능

- 기존 도서 데이터 조회
- 기존 표지 이미지 표시
- 제목 / 작가 / 내용 수정
- API Key 입력
- 이미지 품질 선택
- 이미지 미리보기 재생성
- 수정 완료 시 PATCH 요청
- `updatedAt` 갱신

## 5. 삭제 기능

상세 모달에서 삭제 버튼을 누르면 해당 도서를 삭제합니다.

```http
DELETE /books/:id
```

삭제 전 확인 창을 표시합니다.

## 6. 좋아요 기능

상세 모달에서 좋아요 버튼을 누르면 해당 도서의 좋아요 수가 증가합니다.

```http
PATCH /books/:id
```

## 7. 조회수 기능

도서 카드를 클릭하여 상세 모달을 열면 조회수가 증가합니다.

```http
PATCH /books/:id
```

## 8. 빈 상태 처리

다음 상황에서 안내 문구를 표시합니다.

| 상황 | 안내 문구 |
|---|---|
| 등록된 도서 없음 | 등록된 도서가 없습니다. |
| 검색 결과 없음 | 검색 결과가 없습니다. |
| Home 인기 도서 없음 | 등록된 도서가 없습니다. |
